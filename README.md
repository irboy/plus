اطلاعات خواسته شده را جایگزین کنید.
فایل را بروی سرور اپلود کرده و دستورات زیر را ترمینال وارد کنید:
cd AdvanPlus
chmod +x lunch
./lunch install
./lunch cli
شماره ربات را وارد کنید.
برای خاموش نشدن ربات و اتولانچ :
cd AdvanPlus
cd Cli
screen ./lunch cli
و برای api
cd AdvanPlus
cd Cli
screen ./lunch api
